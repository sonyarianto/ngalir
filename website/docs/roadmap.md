# Roadmap

All 14 phases are complete. Ngalir is now a production-ready low-code automation platform with:

- ✅ Declarative YAML DAG engine with bounded concurrency
- ✅ 30+ node types (databases, AI, HTTP, file formats, integrations)
- ✅ AI-native flow generation and optimization
- ✅ Web UI with drag-and-drop flow editor
- ✅ Structured credential management with AES-256-GCM encryption
- ✅ OAuth2 authorization flow
- ✅ Execution history with SQLite persistence
- ✅ Prometheus metrics and health endpoints
- ✅ Docker images and one-liner install script
- ✅ Node registry (`ngalir search` / `ngalir install`)
- ✅ Per-node auto-generated documentation

See the full [ROADMAP.md on GitHub](https://github.com/sonyarianto/ngalir/blob/main/docs/ROADMAP.md) for complete phase details.
