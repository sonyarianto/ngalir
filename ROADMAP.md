# Roadmap

## Phase 1: Empty Canvas with "Add First Step" (n8n-like UX)

- [ ] Show empty state when canvas has zero nodes: centered prompt with "Add first step" button
- [ ] Clicking the prompt opens a node-type selector/dialog (filterable list of available nodes)
- [ ] Selecting a node type creates the first node at canvas center and auto-selects it

## Phase 2: Inline Node Picker

- [ ] Clicking "+" between nodes or at canvas edges opens a compact node palette popover
- [ ] Filter/search bar in the picker
- [ ] Auto-connect wires when inserting a node between two connected nodes

## Phase 3: Node Execution Indicators

- [ ] Animated pulse/spinner on nodes during execution
- [ ] Output preview badge on nodes after completion
- [ ] Collapsible inline output panel instead of sidebar-only preview

## Phase 4: UX Polish

- [ ] Keyboard shortcut palette (Cmd+K)
- [ ] Undo/redo toast feedback
- [ ] Auto-save with debounce
- [ ] Node grouping / comment boxes with labels
- [ ] Mini-map for large flows
